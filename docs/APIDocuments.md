# api文档

以下api前缀均为： `/api/v1`


### 一、账户

1. 注册 `/register`

   | username    | 用户名   |
   | ----------- | -------- |
   | email       | 邮箱     |
   | pwd         | 密码     |
   | confirm_pwd | 重复密码 |
   
   
   
2. 登录 `/login`

    | username | 用户名 | 当前字段可以传入用户名或者邮箱 |
    | -------- | ------ | ------------------------------ |
    | pwd      | 密码   |                                |

    

3. 修改密码 `/mPassword`
   
    | username    | 用户名     | 当前字段可以传入用户名或者邮箱 |
    | ----------- | ---------- | ------------------------------ |
    | old_pwd     | 旧密码     |                                |
    | pwd         | 新密码     |                                |
    | confirm_pwd | 重复新密码 |                                |
